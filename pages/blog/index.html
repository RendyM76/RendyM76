Building Corebell Engine: A Journey into 3D Creativity

Hey everyone!
I’ve been diving deep into the world of 3D graphics and web-based engines lately, and I’m thrilled to share some updates on my latest project: Corebell Engine. It’s been an exciting journey, and I’ve been working on some cool new features that I think you’ll love. Let me walk you through what I’ve been up to!

What is Corebell Engine?

Corebell Engine is a lightweight, web-based 3D engine built using Three.js. It’s designed to be a simple yet powerful tool for creating and manipulating 3D scenes directly in the browser. Whether you’re a developer, designer, or just someone who loves tinkering with 3D, Corebell Engine is meant to make 3D creation accessible and fun.

What’s New?

Here’s a sneak peek at some of the latest features I’ve been working on:

1. Movable Widgets

One of the most requested features was the ability to move UI panels around. I’m happy to say that Corebell Engine now supports movable widgets! You can drag the controls panel anywhere on the screen, and its position is saved so it stays where you left it, even after refreshing the page. This makes the interface much more flexible and user-friendly.

2. Object Highlighting

Selecting objects in a 3D scene can sometimes be tricky, so I added object highlighting. When you click on an object, it gets a glowing outline, making it super clear which object you’re working with. It’s a small change, but it makes a big difference in usability.

3. Snap-to-Grid

Precision is key when working with 3D scenes, so I implemented a snap-to-grid feature. Now, you can align objects perfectly with the grid, making it easier to create clean, organized scenes. You can also adjust the grid size and toggle snapping on or off.

4. Undo/Redo

We’ve all been there—accidentally deleting or moving something and wishing we could undo it. Well, now you can! Corebell Engine now supports undo/redo for actions like adding, deleting, and transforming objects. It’s a lifesaver for those “oops” moments.

5. Object Properties Panel

I’ve also added an object properties panel that lets you edit the selected object’s position, rotation, scale, and material color. It’s a great way to fine-tune your scene without having to rely solely on the transform controls.

What’s Next?

I’m just getting started! Here are some of the features I’m planning to work on next:

  • Multi-Select: Select and manipulate multiple objects at once.
  • Improved Lighting: Add more lighting options and controls.
  • Scene Export: Export your scenes to popular formats like .glb or .glTF.
  • Animation System: Create simple animations for objects in your scene.

Why Am I Building This?

I’ve always been fascinated by 3D graphics and the creative possibilities they offer. With Corebell Engine, I wanted to create a tool that’s easy to use but still powerful enough to bring ideas to life. It’s been a great learning experience, and I’m excited to see where it goes from here.

Try It Out!

If you’re curious, you can check out the latest version of Corebell Engine here. I’d love to hear your feedback and suggestions for new features. Feel free to drop a comment or reach out to me directly.

Final Thoughts

Building Corebell Engine has been a rewarding journey, and I’m excited to keep improving it. Whether you’re a seasoned developer or just starting out with 3D, I hope Corebell Engine inspires you to create something amazing. Stay tuned for more updates, and happy creating!
